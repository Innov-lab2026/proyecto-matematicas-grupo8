## Configuración de Supabase (Auth & API)
#### *- Obtenidos en:* Consola Supabase -> Settings -> API
SUPABASE_URL="https://[TU_PROYECTO].supabase.co"
SUPABASE_ANON_KEY="[ANON_KEY]"

### Cadenas de conexión para la webDB
#### *- Obtenidos en:* Consola Supabase -> "CONNECT" *(icono "enchufe")* --> "DIRECT - CONNECTION STRING" --> "TRANSACTION_POOLER" *(para el uso de la "Planta de Tablas")*
DATABASE_URL="postgresql://postgres.[USUARIO]:[CONTRASEÑA]@aws-1-[ZONA].pooler.supabase.com:6543/postgres?pgbouncer=true&connection_limit=1"

#### - Obtenidos en: Consola Supabase -> "CONNECT" (icono "ENCHUFE") --> "DIRECT - CONNECTION STRING" --> "SESSION_POOLER" *(Migración y Mantenimiento - Back-End)*
DIRECT_URL="postgresql://postgres.[USUARIO]:[CONTRASEÑA]@aws-1-[ZONA].pooler.supabase.com:5432/postgres"

### Integración con Gemini AI (Google AI Studio)

#### - Con "CUENTA GOOGLE" --> "https://aistudio.google.com/" --> "ACCEDER" --> "PROYECTOS" *generar un proyecto nuevo, por ejemplo: "MATE+"* --> "GET API KEY" *generar y copiar la Key.*
GOOGLE_API_KEY="[TU_API_KEY]"
